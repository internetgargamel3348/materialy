# Vzorečky

Kolekce vzorečků pro skupinu *Beta* v Matematice

### Krychle
		
+ V = a³
+ S = 6 * a²

### Kvádr

+ V = a * b * c
+ S = 2 * (ab + bc + ac)

### Válec

+ V = π * r² * v 
+ S = 2 * π * r * (r + v)

### Pravidelný čtyřboký jehlan

+ V = (a² * v) / 3
+ S = a² + 2 * a * v [výška strany]

### Kužel

+ V = (π * r² * v) / 3
+ S = π * r * (r + √r² * √v²)
